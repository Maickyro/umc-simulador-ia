# Arquitectura del proyecto

## Objetivo

Desarrollar un simulador de cierre de brechas para la Universidad Corporativa Claro (UMC), capaz de registrar aprendices, evaluar competencias mediante inteligencia artificial y visualizar la evolución del aprendizaje.

## Tecnologías

- Next.js
- React
- TypeScript
- Tailwind CSS
- Gemini API (Google AI)
- LocalStorage

## Arquitectura

La aplicación está dividida en cuatro módulos principales:

1. Registro del aprendiz.
2. Evaluación mediante IA.
3. Dashboard de competencias.
4. Persistencia de información.
